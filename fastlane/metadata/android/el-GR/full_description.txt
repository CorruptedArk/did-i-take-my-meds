Η εφαρμογή Πήρα Τα Φάρμακά Μου; βοηθά να παρακολουθείτε πότε πρέπει να παίρνετε φάρμακα. Έχει σχεδιαστεί για να αποτρέπει τον χρήστη από το να παραλείψει τις δόσεις ή/και την υπερδοσολογία.

Η εφαρμογή Πήρα Τα φάρμακά Μου; επιτρέπει στον χρήστη να αποθηκεύσει πολλά φάρμακα με πολύπλοκα χρονοδιαγράμματα. Κάθε φάρμακο μπορεί να ενεργοποιήσει μια ανεξάρτητη ειδοποίηση υπενθύμισης. Επιπλέον, αυτή η εφαρμογή επιτρέπει στον χρήστη να καταγράφει πότε παίρνει μια δόση. Ως μέρος μιας ρουτίνας φαρμακευτικής αγωγής, αυτή η εφαρμογή μπορεί να βοηθήσει στην αποφυγή υπερδοσολογίας και παράλειψης δόσεων.
