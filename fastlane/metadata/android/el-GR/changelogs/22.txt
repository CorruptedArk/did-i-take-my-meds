v1.4.0

Αλλαγές
- Οι τύποι φαρμάκων εμφανίζονται πλέον στην κύρια λίστα φαρμάκων
- Οι προβολές έχουν αναδιαταχθεί και τροποποιηθεί για να χωρέσουν τις νέες επιλογές
- Οι μεταφράσεις στα γερμανικά έχουν ενημερωθεί από τον jpameli στο GitHub

Προσθήκες
- Προσθήκη επιλογής για να καταστήσετε ένα φάρμακο ανενεργό
- Προσθήκη επιλογής για τον καθορισμό των τύπων φαρμάκων
- Προσθήκη επιλογής για ταξινόμηση κατά τύπους ιατρικής
- Προσθήκη επιλογής για να καθορίσετε τον αριθμό Rx για κάθε φάρμακο
- Προσθήκη επιλογής για τον καθορισμό του μεγέθους της δόσης για κάθε φάρμακο
- Προσθήκη επιλογής για να καθορίσετε το όνομα φαρμακείου για κάθε φάρμακο
- Προσθήκη επιλογής για παρακολούθηση του αριθμού των υπόλοιπων δόσεων για κάθε φάρμακο
- Προστέθηκε η επιλογή για να καθορίσετε εάν κάθε φάρμακο πρέπει να λαμβάνεται με τροφή
- Προστέθηκαν γαλλικές μεταφράσεις από τον Edanas στο Weblate
- Προστέθηκαν εβραϊκές μεταφράσεις από τον Yaron στο Weblate
